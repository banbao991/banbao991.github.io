静态托管网页搭建

0. 一些说明

  • 基于 Hexo 进行搭建
    • https://hexo.io/zh-cn/
  • 需要的组件
    • Git
    • node.js
  • 我的版本信息
部件 版本
node.js v16.18.0
npm 8.19.2
Git git version 2.38.1.windows.1

1. 安装 Hexo

  • 建议全局安装 hexo
    • 其他的插件只需要安装在局部即可
  • 如果下载不下来的话可以换源
  • 需要将 hexo 所在的目录添加到环境变量
    • 安装 npm 的时候应该会自动进行这一步

2. 开始我们的快乐

  • 如果命令太慢的话可以直接对 npm 换源
  • 新建一个空白文件夹
    • 之后的命令行操作均在该文件夹目录下进行
  • 生成页面
  • 预览页面
  • 指定端口打开预览

3. 部署 Hexo 到 GitHub Pages

  • 安装 hexo-deployer-git
    • 这里是安装在局部
  • --save
    • npm 5.0.0 之前，有 --save 参数才会把模块写入到 packages.json
    • 现在不需要写了
  • 修改配置文件 _config.yml 的 Deployment 部分
  • 将 "banbao991" 修改为你自己的用户名即可
  • 上传
  • 没有权限上传
    • 需要有 repo 的访问权限
    • 需要配置本地 git 的 ssh key
      • 配置方法
      • 这里配置好了之后，repo 之后可以使用 git 而不是 http 下载，不易被墙
